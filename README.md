# X 未互关检查助手

一个 Chrome / Edge 本地扩展，用于扫描 X/Twitter 的“正在关注”列表，找出当前没有在用户名附近显示 `关注了你` 的账号。

## 功能

- 从账号主页读取“正在关注”人数。
- 自动进入 `/following` 页面并滚动扫描。
- 判断每个账号的 `@用户名` 附近是否显示 `关注了你` / `Follows you`。
- 只显示“当前未互关”名单。
- 支持打开主页复核、导出 JSON、清空本地数据。
- 数据只保存在浏览器本地，不上传服务器。

## 安装

1. 打开 Chrome 或 Edge。
2. 进入 `chrome://extensions` 或 `edge://extensions`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目文件夹：`x-unfollow-helper`。

## 使用教程

### 1. 打开账号主页

进入你的 X/Twitter 账号主页，例如：

```text
https://x.com/你的用户名
```

插件会从主页读取“正在关注”的人数，然后再进入 following 页面扫描。

### 2. 从主页校准扫描

点击扩展里的 **从主页校准扫描**。

插件会自动执行：

1. 打开账号主页。
2. 读取“正在关注”人数。
3. 自动跳转到 `/following`。
4. 从顶部开始滚动扫描。
5. 保存当前未互关名单。

### 3. 判断标准

插件只看 `@用户名` 附近有没有 `关注了你` / `Follows you`。

- 有 `关注了你`：说明对方也关注你，不进入未互关名单。
- 没有 `关注了你`：说明当前没有看到对方关注你，进入未互关名单。
- 右侧 `正在关注` 只代表你关注了对方，不代表互关。
- 简介里的文字不参与判断。

### 4. 查看结果

扫描完成后，插件只显示 **当前未互关**。

你可以勾选账号，点击 **打开所选**，到 X 页面手动复核并决定是否继续关注。

### 5. 导出与清空

需要排查时可以点击 **导出 JSON**。

所有数据都保存在浏览器本地 `chrome.storage.local`，可以随时清空。

## 隐私说明

- 不读取 GitHub、X/Twitter 或任何网站的密码。
- 不调用非公开接口。
- 不自动点击关注或取关按钮。
- 不上传任何账号列表到远程服务器。
- 扫描数据只保存在本地浏览器。

## 已知限制

- X/Twitter 的网页结构经常变化，扩展需要随页面变化维护。
- X 的虚拟列表可能会重复加载、漏加载或短暂停止加载。
- 如果页面关注人数与快照数量有差异，插件会保存扫到的结果并提示“未完全校准”。
- 如果本次快照为 0，插件会拒绝覆盖旧数据。
